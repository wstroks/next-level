<h2 title='#projeto'>
    Projeto ECOLETA
</h2>

Implementação de um marketplace de coleta de resíduos, para ajudar pessoas a encontrar pontos de coleta de forma eficiente.

<h2 title='#tecnologias'>
    O que foi Utilizado 
</h2>

Esse projeto foi desenvolvido com as seguintes tecnologias:

- Nodejs
- Reactjs
- React Native
- Typescript
